# Vert 0.1.0-beta
A compiled language created to expand compatibility with JS

## How to use

- With **Vert.js** File
```
node vert.js yourfile.vert
```

- With **Vert.exe** File
```
vert yourfile.vert
```

> Running Vert without any arguments starts the interactive mode, where the Vert code you enter is compiled to JavaScript in real time and executed immediately

**For a detailed grammar explanation, refer to SYNTAX.md**
